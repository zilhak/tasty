# 21. 복사 모드

## cmux 구현 방식

- vi 스타일 네비게이션으로 스크롤백 탐색
- 셀 단위 선택
- Vim 모드 인디케이터

## 크로스 플랫폼 구현 방안

### 구현 방법

GUI 앱이므로 두 가지 복사 방식을 모두 지원:

1. **마우스 선택**: 드래그로 텍스트 선택 → 활성화된 복사/붙여넣기 방식에 따라 복사 (세 가지 키 방식은 [10-keyboard-shortcuts.md](10-keyboard-shortcuts.md) 참조)
2. **키보드 복사 모드**: vi 스타일로 정밀한 셀 단위 선택

### 마우스 선택

| 동작 | 기능 |
|------|------|
| 클릭 + 드래그 | 텍스트 선택 (셀 단위) |
| 더블클릭 | 단어 선택 |
| 트리플클릭 | 줄 선택 |
| Shift+클릭 | 선택 영역 확장 |
| Ctrl+클릭 (Cmd+클릭) | URL 열기 |
| 드래그 중 스크롤 | 스크롤백 영역까지 선택 확장 |

네이티브 GUI이므로 마우스 이벤트를 완벽하게 제어할 수 있다.

### 키보드 복사 모드

vi 스타일 키보드 네비게이션:

1. 복사 모드 진입 (단축키, 예: Ctrl+Shift+C 또는 설정 가능)
2. vi 키로 커서 이동
3. Space/v로 선택 시작
4. Enter/y로 선택 복사 (→ 클립보드)
5. q 또는 ESC로 종료

### 키 바인딩

| 키 | 동작 |
|----|------|
| h/j/k/l | 커서 이동 |
| w/b | 단어 단위 이동 |
| 0/$ | 줄 시작/끝 |
| g/G | 버퍼 시작/끝 |
| Ctrl+U/D | 반 페이지 스크롤 |
| / | 검색 |
| n/N | 다음/이전 검색 결과 |
| Space/v | 선택 시작 (문자 단위) |
| V | 줄 선택 모드 |
| Ctrl+V | 블록 선택 모드 (사각형) |
| y/Enter | 선택 복사 |
| q/ESC | 종료 |

### GPU 렌더링 선택 표시

| 요소 | 렌더링 |
|------|--------|
| 선택 영역 | 반투명 하이라이트 오버레이 (셰이더) |
| 커서 (복사 모드) | 블록 커서, 깜빡임 |
| 모드 인디케이터 | 하단 상태바에 "COPY MODE" / "VISUAL" / "VISUAL LINE" 표시 |
| 검색 매치 | 추가 하이라이트 색상 |

### 클립보드 연동

선택 텍스트는 `arboard` 크레이트로 OS 클립보드에 직접 복사.
설정에 따라 선택 즉시 복사 옵션 제공 (select-to-clipboard).

## 최적화 전략

- **선택 영역 렌더링**: 선택 상태가 변경된 셀만 재렌더링한다. 선택 범위의 시작/끝이 바뀐 행만 다시 그린다.
- **대량 복사 스트리밍**: 대량 텍스트 선택 시 메모리에 전부 올리지 않고 스트리밍으로 클립보드에 전달한다. 수만 줄 선택에서도 메모리 사용량을 일정하게 유지한다.
- **vi 모드 키 처리 빠른 경로**: 복사 모드에서는 일반 키맵 조회를 스킵하고, 복사 모드 전용 키 핸들러로 바로 라우팅한다. 불필요한 바인딩 탐색을 제거한다.

## 구현 가능 여부

| OS | 가능 여부 |
|----|-----------|
| Windows | ✅ |
| macOS | ✅ |
| Linux | ✅ |

GPU 렌더링 + 직접 클립보드 접근. cmux보다 더 풍부한 선택/복사 UX가 가능하다.
